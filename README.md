## Features
- Playable for 2 players (X and O)
- Shows winner or draw
- Reset / New Game button
- Simple and clean UI
---
## How to Play
- Player **X** plays first.
- Click on any box to place X or O.
- First to make 3 in a row (line, column, diagonal) wins.
